**Набор из 5 консольных мини-игр** для тренировки внимания и логики:

- **brain-even** — «Чётное ли число?»
- **brain-calc** — «Вычислите выражение»
- **brain-gcd** — «Найдите НОД»
- **brain-progression** — «Какое число пропущено?»
- **brain-prime** — «Простое ли число?»

---

## Требования

- **PHP** ≥ 7.4
- **Composer**

---

## Установка

```bash
git clone git@github.com:eager94/php-project-45.git
cd php-project-45
composer install
```
Для запуска отдельных игр make + название игры
```
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```
